# 蚂蚁森林自动收能量脚本

使用 uiautomator2 操作手机解锁, 打开支付宝, 收取能量

## 1. 环境:
    python版本: 3.10
    opencv-python==4.11.0.86
    numpy==1.26.4
    uiautomator2==2.16.25

## 2. 快速上手
    1. 环境配置
      1. 手机打开 USB调试、 USB安装应用（自动安装ATX软件）、显示指针位置（用于获取九宫格坐标）
      2. 修改settings中的九宫格坐标及密码
    2. 验证环境， 运行test.py，验证是否打开支付宝，然后锁屏
    3. 直接运行 main.py
